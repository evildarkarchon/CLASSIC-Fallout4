**lib.rs migration:**
1. Replace `use once_cell::sync::Lazy;` (line 16) with `use std::sync::LazyLock;`
2. Change `static RUNTIME: Lazy<Runtime> = Lazy::new(|| {` to `static RUNTIME: LazyLock<Runtime> = LazyLock::new(|| {`
3. The API is identical (both use `.new(|| ...)` with same closure), so no other code changes needed

**performance_core.rs migration:**
1. Replace `use once_cell::sync::Lazy;` (line 7) with `use std::sync::LazyLock;`
2. Change `static METRICS: Lazy<Arc<PerformanceMetrics>> = Lazy::new(|| ...);` to `static METRICS: LazyLock<Arc<PerformanceMetrics>> = LazyLock::new(|| ...);`
3. Change `static TIMER_START: Lazy<Instant> = Lazy::new(Instant::now);` to `static TIMER_START: LazyLock<Instant> = LazyLock::new(|| Instant::now());`
   NOTE: LazyLock requires a closure, not a function pointer. `Lazy::new(Instant::now)` works but `LazyLock::new(Instant::now)` may not -- use `LazyLock::new(|| Instant::now())` to be safe.
