**Create `settings_general.slint`:**

Component `SettingsGeneral` with these properties and controls:
- `in-out property <int> game-version-index: 0;` -- ComboBox index (0=Auto, 1=Original, 2=NextGen, 3=VR)
- `in-out property <string> auto-detected-hint: "";` -- hint text like "(detected: NextGen)"
- `in-out property <bool> update-check: true;`
- `in-out property <int> update-source-index: 0;` -- ComboBox index (0=GitHub, 1=Both)
- `in-out property <bool> fcx-mode: false;`
- Callbacks: `callback game-version-changed(int);`, `callback update-check-changed(bool);`, `callback update-source-changed(int);`, `callback fcx-mode-changed(bool);`

Layout: VerticalBox with padding 16px, spacing 12px. Each setting is a HorizontalBox with:
- Text label (min-width: 160px, vertical-alignment: center)
- Control (ComboBox or Switch)

Game Version row: ComboBox model `["Auto", "Original", "NextGen", "VR"]` with `current-index <=> root.game-version-index`. On `selected()`, call `root.game-version-changed(self.current-index)`. After the ComboBox, conditionally show hint text when `game-version-index == 0`: `Text { text: root.auto-detected-hint; color: #888888; font-size: 11px; vertical-alignment: center; }`.

Update Check row: Switch with `checked <=> root.update-check`. Use the Switch's `toggled` callback to fire `root.update-check-changed(self.checked)`.

Update Source row: ComboBox model `["GitHub", "Both"]` with `current-index <=> root.update-source-index`. On `selected()`, call `root.update-source-changed(self.current-index)`.

FCX Mode row: Switch with `checked <=> root.fcx-mode`. Use `toggled` callback to fire `root.fcx-mode-changed(self.checked)`.

Import `{ Switch, ComboBox, VerticalBox, HorizontalBox, Text }` from "std-widgets.slint". Note: Slint std-widgets may not export `Text` directly -- `Text` is a built-in element, not imported. Only import actual std-widgets components.

**Important Slint notes:**
