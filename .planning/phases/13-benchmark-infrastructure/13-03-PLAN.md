Create Python script to extract p95/p99 percentiles from Criterion raw.csv files:

1. Create `scripts/bench/extract_percentiles.py`:

Script requirements:
- Find all `target/criterion/*/new/raw.csv` files
- For each CSV, calculate per-iteration times: `sample_measured_value / iteration_count`
- Calculate percentiles: p50, p95, p99, min, max, mean, stddev
- Output to `target/criterion/percentiles.json`
- Also print human-readable summary to stdout

Implementation:
```python
#!/usr/bin/env python3
"""Extract p95/p99 percentiles from Criterion raw.csv files.

Criterion natively reports mean/median/MAD but not tail percentiles.
This script post-processes raw.csv to calculate p50/p95/p99.

Usage:
    python extract_percentiles.py [--output FILE]

Output format (JSON):
{
    "benchmark_name": {
        "min": 1234.5,
        "p50": 1300.0,
        "p95": 1450.0,
        "p99": 1500.0,
        "max": 1600.0,
        "mean": 1310.5,
        "stddev": 45.2,
        "unit": "ns"
    }
}
"""
```

Key functions:
- `calculate_percentiles(values: list[float]) -> dict`
- `process_raw_csv(csv_path: Path) -> dict`
- `main()` - Find CSVs, process, output JSON

Use standard library only (csv, json, pathlib, math, statistics).
No external dependencies required.

Handle edge cases:
- Empty CSV files
- Missing directories
- Invalid data formats

Create Python script to manage baseline storage:

- Keep the 10 most recent baselines
- Delete older baselines
- Dry-run mode by default (show what would be deleted)
- `--execute` flag to actually delete

Implementation:
```python
#!/usr/bin/env python3
"""Clean up old Criterion baselines, keeping the 10 most recent.

Baselines are identified by naming pattern: baseline-YYYY-MM-DD-HHMMSS
Non-matching directories are left untouched.

Usage:
    python cleanup_baselines.py           # Dry run - show what would be deleted
    python cleanup_baselines.py --execute # Actually delete old baselines
    python cleanup_baselines.py --keep 5  # Keep only 5 most recent
"""
```

Key functions:
- `find_baselines(criterion_dir: Path) -> list[tuple[datetime, Path]]`
- `cleanup_baselines(keep: int, execute: bool) -> None`
- `main()` with argparse for CLI

Parse timestamp from directory name for sorting.
Print clear output showing what's kept vs deleted.
