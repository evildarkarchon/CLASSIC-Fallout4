  <action>
Create PowerShell script for running benchmarks with baseline management:

1. Create `scripts/bench/` directory

2. Create `scripts/bench/run_benchmarks.ps1` with parameters:
   - `-Mode` [quick|thorough] - Sets BENCH_MODE env var (default: quick)
   - `-SaveBaseline` - Save results as baseline with timestamp name
   - `-BaselineName` - Override automatic timestamp naming
   - `-Compare` - Compare against specified baseline
   - `-Crate` - Run benchmarks for specific crate only (optional)

3. Script logic:
   - Validate release mode (cargo bench implies --release)
   - Set BENCH_MODE environment variable
   - Run `cargo bench` in rust/ directory
   - If -SaveBaseline: use timestamp format `baseline-yyyy-MM-dd-HHmmss`
   - If -Compare: pass `--baseline $BaselineName` to cargo bench
   - Print summary of what was run

4. Add usage documentation in script header:
   ```powershell
   # Examples:
   #   .\run_benchmarks.ps1                     # Quick mode, no baseline
   #   .\run_benchmarks.ps1 -Mode thorough      # Thorough mode
   #   .\run_benchmarks.ps1 -SaveBaseline       # Save baseline with timestamp
   #   .\run_benchmarks.ps1 -Compare -BaselineName "baseline-2026-02-04-120000"
   ```

Note: Script should work from project root directory.
  </action>
