<objective>
Collapse the multi-layer factory/detector/status architecture into a single flat factory.py module.

Purpose: Eliminate the three-layer detection system (detector -> factory/core -> status) that caches and tracks component availability redundantly. Python's sys.modules already handles import caching, making the custom layers pure overhead.

Output: A single `ClassicLib/integration/factory.py` module that re-exports all 23+ factory functions with direct try-import pattern, plus `detect_component()` utility. The factory/ subpackage, detector.py, and status.py are deleted. All callers updated.
</objective>
