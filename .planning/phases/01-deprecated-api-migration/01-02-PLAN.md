### parser.rs -- DEBT-05 (per D-01, D-02, D-03)

1. Add `use pyo3::exceptions::PyDeprecationWarning;` to imports at top of file (after existing `use pyo3::prelude::*;` line).

2. Replace the `parse_segments_parallel` method (lines 96-122) entirely. The new version:
   - Removes `#[allow(deprecated)]` annotation
   - Changes return type from `PyResult<Bound<'py, PyList>>` to `PyResult<Bound<'py, PyDict>>`
   - Emits `DeprecationWarning` via `PyErr::warn` with message: `c"parse_segments_parallel is deprecated. Use parse_all_sections instead, which returns dict[str, list[str]]."`
   - Calls `self.inner.parse_all_sections_arc(&arc_lines)` instead of `self.inner.parse_segments_parallel`
   - Converts HashMap result to PyDict (follow the parse_complete pattern at lines 226-235, but do NOT call `.trim()` on lines -- the parse_complete trims but parse_segments_parallel should preserve original whitespace)
   - The `chunk_size` parameter is kept in the signature (with underscore prefix `_chunk_size`) for backward compat but ignored

The replacement method body:
```rust
#[pyo3(name = "parse_segments_parallel", signature = (lines, chunk_size=None))]
pub fn parse_segments_parallel<'py>(
    &self,
    py: Python<'py>,
    lines: Vec<String>,
    _chunk_size: Option<usize>,
) -> PyResult<Bound<'py, PyDict>> {
    PyErr::warn(
        py,
        &py.get_type::<PyDeprecationWarning>(),
        c"parse_segments_parallel is deprecated. Use parse_all_sections instead, which returns dict[str, list[str]].",
        1,
    )?;
    let arc_lines = to_arc_str_vec(&lines);
    let named_sections = without_gil(py, || {
        self.inner.parse_all_sections_arc(&arc_lines)
    });
    let dict = PyDict::new(py);
    for (key, section_lines) in &named_sections {
        let py_lines: Vec<&str> = section_lines.iter().map(|s| s.as_ref()).collect();
        let inner_list = PyList::new(py, py_lines)?;
        dict.set_item(key.as_str(), inner_list)?;
    }
    Ok(dict)
}
```

The replacement method body:
```rust
/// Generate suspect section (legacy method for backward compatibility)
pub fn generate_suspect_section(&self, py: Python<'_>, found_suspects: Vec<String>) -> PyResult<PyReportFragment> {
    PyErr::warn(
        py,
        &py.get_type::<PyDeprecationWarning>(),
        c"generate_suspect_section is deprecated. Use generate_suspect_section_header and generate_suspect_found_footer instead.",
        1,
    )?;
    let header = self.inner.generate_suspect_section_header();
    let found_suspect = !found_suspects.is_empty();
    let footer = self.inner.generate_suspect_found_footer(found_suspect);
    Ok(PyReportFragment { inner: header.combine(&footer) })
}
```
