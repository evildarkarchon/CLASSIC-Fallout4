  <name>Task 1: Create shared signing script and update .gitignore</name>
  <files>tools/sign-binaries.ps1, .gitignore</files>
  <action>
Create `tools/sign-binaries.ps1` with four functions:

1. **Find-SignTool** -- Locates signtool.exe. Check PATH first (`Get-Command signtool.exe -ErrorAction SilentlyContinue`), then read `HKLM:\SOFTWARE\Microsoft\Windows Kits\Installed Roots` for `KitsRoot10`, then glob `$sdkRoot\bin\*\x64\signtool.exe` sorted descending by version. Return `$null` if not found.

2. **Ensure-SigningCert** -- Parameters: `$PfxPath`, `$Password = "classic-dev-signing"`, `$RenewalWindowDays = 30`, `$ValidityYears = 1`. Logic:
   - If PFX does not exist -> generate
   - If PFX exists -> load via `[X509Certificate2]::new($PfxPath, $secPassword)`, check `($cert.NotAfter - (Get-Date)).Days`, dispose cert. If <= 30 days -> regenerate
   - If PFX exists and > 30 days -> log "valid for N more days" and return
   - Generation: `New-SelfSignedCertificate -Type CodeSigningCert -Subject "CN=CLASSIC Dev Signing, O=CLASSIC Project" -FriendlyName "CLASSIC Code Signing" -CertStoreLocation "Cert:\CurrentUser\My" -NotAfter (Get-Date).AddYears($ValidityYears) -KeyExportPolicy Exportable -KeyAlgorithm RSA -KeyLength 2048 -HashAlgorithm SHA256`. Create parent directory if needed. Export with `Export-PfxCertificate -CryptoAlgorithmOption AES256_SHA256`. Remove from cert store immediately after export (`Remove-Item -Path "Cert:\CurrentUser\My\$($cert.Thumbprint)"`). Use `Out-Null` to suppress output (NEVER redirect to `nul`).

3. **Sign-Binary** -- Parameters: `$SignToolPath`, `$PfxPath`, `$Password`, `$BinaryPath`. Check binary exists with `Test-Path`. Run `& $SignToolPath sign /f $PfxPath /p $Password /fd SHA256 /d "CLASSIC" $BinaryPath`. Check `$LASTEXITCODE`. Return `$true`/`$false`. Log with `Write-Host` (green for success, warning for failure).

4. **Invoke-CodeSigning** -- Parameters: `$InstallDir`, `$Binaries` (string array of EXE filenames to sign). This is the entry point called by build scripts. Logic:
   - Call `Find-SignTool`. If null -> write "signtool.exe not found. Skipping code signing." in DarkGray and return.
   - Resolve PFX path: `$env:CLASSIC_SIGNING_PFX` if set, else `tools/certs/classic-signing.pfx` relative to the repo root (use `$PSScriptRoot` to derive).
   - Call `Ensure-SigningCert`. If it throws -> write warning and return (do not block builds).
   - For each binary name in `$Binaries`: construct full path as `Join-Path $InstallDir $binary`, call `Sign-Binary`.
   - Wrap the entire function body in try/catch so signing failures never block the build.

IMPORTANT: Never use `2>$null` or redirect to `nul`. Use `-ErrorAction SilentlyContinue` or `Out-Null` patterns.

Add to `.gitignore` at the end of the "User Files" section (before the `# Rust` comment around line 107):
```
# Code signing certificates (auto-generated, contains private key)
tools/certs/
```
